벤더 프리픽스?

주요 웹 브라우저(익스플로러, 크롬, 파이어폭스, 사파리, 오페라 등) 공급자가 새로운 실험적인 기능을 제공할 때 이전 버전의 웹 브라우저에 그 사실을 알려주기 위해 사용하는 접두사(prefix)를 의미.

벤더 프리픽스를 사용할 때?

아직 CSS 권고안에 포함되지 못한 기능이나, CSS 권고 안에는 포함되어 있지만 아직 완벽하게 제정된 상태가 아닌 기능을 사용하고자 할 때!

벤더 프리픽스를 사용하게 되면?

해당 기능이 포함되어 있지 않은 이전 버전의 웹 브라우저에서도 그 기능을 사용할 수 있음.

각 브라우저별 벤더 프리픽스(접두어, 브라우저)
  • -webkit-, 구글과 사파리 브라우저
  • -moz-, 파이어폭스 브라우저
  • 0-, 페라 브라우저
  • -ms-, 익스플로러(대부분 생략함)

  • 스크립트 사용하는 방법 중 가장 간단하게 벤더 프리픽스를 붙여주는 사이트.
  • 사이트에 접속하여 스크립트를 다운로드 후 HTML head에 연결.
    • Prefix-free

복사-붙여넣기

  • CSS 코드를 붙여 넣으면 자동으로 벤더 프리픽스를 붙여주는 사이트.
    • Autoprefixer CSS online
